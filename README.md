# GPA Calculator - Academic Lab

This project demonstrates the development of a GPA calculation system through four progressive stages, moving from basic PHP to dynamic AJAX.

## Project Phases
* Step 1: Basic HTML structure with a separate PHP processing file.
* Step 2: Merged PHP and HTML logic into a single file for seamless interaction.
* Step 3: Implemented AJAX (jQuery) to calculate GPA without reloading the page.
* Step 4: Final UI improvements and dynamic feedback based on results.

## Tech Stack
* Web: HTML5, CSS3, JavaScript (jQuery).
* Backend: PHP.
* Format: JSON.
